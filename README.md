# 📚 Introduction

An Icon package for [QWeather](https://github.com/qwd/Icons), which helps you show its icon continently.

# 📦 How to use

1. Add to `pubspec.yaml`

    ```yaml
    dependencies:
        qweather_icons: ^0.0.1
    ```

2. Use it any where

    ```dart
    Icon(QweatherIcons.tag_qweather),
    ```
